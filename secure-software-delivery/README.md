# Secure Software Delivery — Google Cloud Skill Badge

This folder documents the labs I completed for the **Secure Software Delivery** skill badge (Intermediate level, Security track) as part of the Google Arcade Facilitator Program 2026.

## What this badge covers
This badge is about building a secure CI/CD pipeline on Google Cloud — from writing hardened Dockerfiles, to scanning container images during builds, to gating deployments so only verified/attested images ever reach production (using Binary Authorization).

## Labs completed

| # | Lab | Focus |
|---|-----|-------|
| 1 | [Securing Container Builds](./01-securing-container-builds) | Writing secure Dockerfiles, vulnerability scanning |
| 2 | [Secure Builds with Cloud Build](./02-secure-builds-with-cloud-build) | Automated build pipelines, build triggers |
| 3 | [Gating Deployments with Binary Authorization](./03-gating-deployments-binary-authorization) | Attestors, signing policies, deployment gates |
| 4 | [Secure Software Delivery: Challenge Lab](./04-secure-software-delivery-challenge-lab) | End-to-end pipeline combining all of the above |

Each lab folder has its own README with what I did, the key commands/configs, and what I learned.
